# 💳 Credit Card Transaction Dashboard | Power BI

## 📌 Project Overview
This project is an interactive Credit Card Transaction Dashboard developed using Microsoft Power BI. It helps analyze customer spending behavior, transaction trends, revenue, and interest earned through dynamic visualizations and KPIs. The dashboard enables businesses to make data-driven decisions by providing clear insights into customer and transaction performance.

---

## 🎯 Project Objectives
- Analyze credit card transaction data.
- Track revenue and transaction trends.
- Identify customer spending patterns.
- Compare quarterly business performance.
- Provide interactive reports using filters and slicers.

---

## 🚀 Key Features
- 📊 Revenue Analysis
- 💳 Credit Card Transaction Analysis
- 💰 Interest Earned Analysis
- 👥 Customer Segmentation
- 📈 Quarterly Performance Analysis (Q1–Q4)
- 📅 Weekly Revenue & Transaction Trends
- 🎓 Education-wise Customer Analysis
- 💼 Job-wise Customer Analysis
- 💵 Income Group Analysis
- 🎛️ Interactive Filters & Slicers
- 📌 KPI Cards

## 🛠️ Tools & Technologies

## 📊 Key Performance Indicators (KPIs)
